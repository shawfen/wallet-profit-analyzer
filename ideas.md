# 钱包盈利分析工具 - 设计方案

## 项目概述
一个本地工具，用于导入钱包交易数据表格，自动筛选出盈利倍数大于指定阈值的钱包地址，并按指定格式输出结果。

## 方案一：赛博朋克数据终端风格

<probability>0.08</probability>

<text>
### Design Movement
**Cyberpunk Terminal Aesthetic** - 融合80年代复古终端与未来科技感，营造专业的加密货币/区块链数据分析氛围。

### Core Principles
1. **高对比度视觉层次** - 深色背景配合霓虹色高亮，数据一目了然
2. **信息密度优先** - 紧凑布局，最大化数据展示效率
3. **动态反馈** - 数据处理时的扫描线、闪烁效果增强交互感
4. **工具属性明确** - 界面服务于功能，不做无意义装饰

### Color Philosophy
- **主背景**: 深邃的墨黑色 (#0a0a0f) - 模拟终端屏幕
- **主要文字**: 冷白色 (#e0e0e0) - 高可读性
- **强调色-盈利**: 霓虹绿 (#00ff88) - 代表盈利、成功
- **强调色-高倍数**: 电光蓝 (#00d4ff) - 代表高价值数据
- **警告色**: 琥珀橙 (#ff9500) - 引起注意但不刺眼
- **边框/分隔**: 暗灰色 (#1a1a2e) 配合微光效果

### Layout Paradigm
- **单页工具布局** - 左侧为输入控制面板，右侧为结果展示区
- **卡片式数据块** - 每个功能区域用带发光边框的卡片包裹
- **固定顶部状态栏** - 显示当前处理状态和统计信息
- **底部操作区** - 复制、导出等操作按钮

### Signature Elements
1. **发光边框** - 卡片边框带有微弱的霓虹发光效果
2. **扫描线动画** - 数据加载时的横向扫描线效果
3. **等宽字体数据** - 钱包地址使用等宽字体，增强专业感

### Interaction Philosophy
- 拖拽上传文件时边框发光增强
- 数据筛选时有快速的闪烁反馈
- 复制成功时有短暂的脉冲动画
- 鼠标悬停数据行时背景微亮

### Animation
- 页面加载时元素依次淡入（stagger effect）
- 数据处理中的脉冲呼吸动画
- 结果出现时的打字机效果
- 按钮点击的涟漪扩散效果

### Typography System
- **标题**: JetBrains Mono Bold - 科技感等宽字体
- **正文**: Inter Medium - 现代无衬线，高可读性
- **数据/代码**: JetBrains Mono Regular - 等宽对齐
- **层级**: 24px/18px/14px/12px

## 方案二：极简数据仪表盘风格

<text>
### Design Movement
**Swiss Minimalism + Dashboard** - 瑞士设计的精确与现代仪表盘的功能性结合，强调信息的清晰传达。

### Core Principles
1. **留白即设计** - 大量留白让数据呼吸
2. **网格系统** - 严格的12列网格确保对齐
3. **功能即美学** - 每个元素都有明确目的
4. **层级分明** - 通过大小、颜色、间距建立视觉层次

### Color Philosophy
- **主背景**: 纯净白 (#ffffff) - 最大化可读性
- **次级背景**: 浅灰 (#f8f9fa) - 区分层级
- **主要文字**: 深灰 (#1a1a1a) - 柔和但清晰
- **主强调色**: 深蓝 (#2563eb) - 专业、可信
- **成功色**: 翠绿 (#10b981) - 盈利标识
- **边框**: 极淡灰 (#e5e7eb) - 若隐若现

### Layout Paradigm
- **垂直流式布局** - 从上到下的自然阅读顺序
- **卡片分组** - 输入区、设置区、结果区明确分离
- **响应式栅格** - 大屏双栏，小屏单栏
- **固定操作栏** - 底部固定的操作按钮组

### Signature Elements
1. **圆角卡片** - 统一的16px圆角，柔和专业
2. **微阴影** - 极淡的阴影营造层次感
3. **图标点缀** - 简洁的线性图标辅助理解

### Interaction Philosophy
- 悬停时卡片微微上浮
- 输入框聚焦时边框变色
- 按钮点击有轻微下沉感
- 状态变化有平滑过渡

### Animation
- 300ms的ease-out过渡曲线
- 卡片hover时translateY(-2px)
- 结果列表淡入动画
- 加载状态的骨架屏效果

### Typography System
- **标题**: Plus Jakarta Sans Bold - 现代几何感
- **正文**: Inter Regular - 中性可读
- **数据**: SF Mono / Menlo - 等宽数字
- **层级**: 28px/20px/16px/14px

## 方案三：暗色金融终端风格

<text>
### Design Movement
**Bloomberg Terminal Inspired** - 借鉴专业金融终端的设计语言，营造高端、专业的数据分析工具感。

### Core Principles
1. **数据密度** - 在有限空间内展示最多有效信息
2. **色彩编码** - 用颜色快速区分数据类型和状态
3. **操作效率** - 减少点击，提升工作流效率
4. **专业可信** - 设计传达工具的可靠性

### Color Philosophy
- **主背景**: 深海蓝黑 (#0d1117) - 长时间使用不疲劳
- **卡片背景**: 深灰蓝 (#161b22) - 层次分明
- **主要文字**: 银白色 (#c9d1d9) - 柔和不刺眼
- **高亮数据**: 金色 (#f0b429) - 重要数据醒目
- **盈利色**: 明亮绿 (#3fb950) - 正向指标
- **边框**: 深边框 (#30363d) - 结构清晰

### Layout Paradigm
- **工具栏+工作区** - 顶部工具栏，下方主工作区
- **左右分栏** - 左侧控制面板(30%)，右侧结果区(70%)
- **表格为主** - 结果以紧凑表格形式呈现
- **状态指示器** - 实时显示处理进度和统计

### Signature Elements
1. **金色强调** - 关键数据用金色高亮
2. **紧凑表格** - 高密度数据表格设计
3. **状态徽章** - 小型彩色徽章标识状态

### Interaction Philosophy
- 表格行悬停高亮
- 快捷键支持（Ctrl+V粘贴）
- 一键复制结果
- 实时筛选预览

### Animation
- 最小化动画，追求响应速度
- 数据更新时的快速闪烁
- 进度条的流畅推进
- 成功/错误的短暂提示

### Typography System
- **标题**: IBM Plex Sans Bold - 专业科技感
- **正文**: IBM Plex Sans Regular - 一致性
- **数据**: IBM Plex Mono - 等宽数据展示
- **层级**: 20px/16px/14px/12px

## 选定方案

**选择方案三：暗色金融终端风格**

理由：
1. 与加密货币/区块链数据分析的专业场景高度契合
2. 深色背景适合长时间数据分析工作
3. 金色强调色能有效突出盈利数据
4. 高数据密度设计符合工具的实用性需求
5. 专业的金融终端风格增强用户信任感
